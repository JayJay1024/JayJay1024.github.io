transaction declares authority错误

2019-02-18

概述：

push action 或者 transaction 的时候，出现类似以下错误：

原因：

检查部署myaccount111的合约代码，看看是不是使用了内联 action 或 defer ，这需要你的myaccount111具有eosio.code权限

解决：

给myaccount111赋予eosio.code权限
